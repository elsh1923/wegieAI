# Amharic Video Captioning Frontend

A premium Next.js interface for uploading videos and downloading Amharic captions.

## Setup

1. Navigate to the frontend directory:

   ```bash
   cd frontend
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Run the development server:

   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features

- **Premium UI**: Dark mode with glassmorphism and smooth animations.
- **Drag & Drop**: Easy video upload via `react-dropzone`.
- **Status Polling**: Real-time updates on transcription progress.
- **Overlay Support**: Option to embed captions directly into the video.
- **Amharic Optimized**: Interface tailored for Ethiopian language transcription.
